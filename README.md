Task 9: Поиск непрерывной возрастающей последовательности максимальной длины с максимальной суммой

Описание

Данный проект реализует алгоритм поиска непрерывной возрастающей последовательности чисел в одномерном массиве на языке C++.

Программа находит непрерывную возрастающую последовательность максимальной длины. Если таких последовательностей несколько, выбирается та, у которой сумма элементов максимальна.

Параметры командной строки

Отсутствуют.

Входные данные

Массив целых чисел, заканчивающийся переходом на новую строку (или EOF). Числа разделяются пробелами.

Выходные данные

Найденная последовательность чисел, разделённая пробелами.

Пример запуска

# ./task9

<1 2 3 0 4 5 6 0

>0 4 5 6

Другие примеры

Вход: 1 2 3 4 5

Выход: 1 2 3 4 5

Вход: 5 4 3 2 1

Выход: 5

Вход: 1 2 3 1 2 3 4 5

Выход: 1 2 3 4 5

Вход: 1 2 3 5 4 5 6 7

Выход: 4 5 6 7

Функциональные требования

Возрастающая последовательность: каждый следующий элемент должен быть строго больше предыдущего (не равно и не больше или равно).

Непрерывность: последовательность должна идти подряд в исходном массиве без пропусков.

Критерии выбора:
- приоритет 1: максимальная длина последовательности
- приоритет 2: при одинаковой длине выбирается последовательность с максимальной суммой элементов

Пустой массив: при пустом входном массиве программа не выводит ничего.

Технические подробности реализации

Структура данных: для хранения массива чисел используется std::vector<int>.

Чтение ввода: программа считывает строку через std::getline, затем с помощью std::stringstream разбивает её на отдельные числа.

Алгоритм: однопроходный (сложность O(n)):
- итераторы it_cur (текущий элемент) и it_st (начало текущей возрастающей последовательности)
- при обнаружении возрастания (it_cur < it_next) текущая сумма увеличивается
- при нарушении возрастания начинается новая последовательность с it_next
- после каждого шага проверяется, лучше ли текущая последовательность найденной ранее (сравнение длины, затем суммы)
- используется std::distance для вычисления длины последовательности по итераторам

Итераторы:
- numbers.begin(): итератор на первый элемент
- numbers.end(): итератор на позицию после последнего элемента (не разыменовывается)
- std::next(it): возвращает итератор, сдвинутый на 1 позицию вперёд
- std::prev(it): возвращает итератор, сдвинутый на 1 позицию назад
- std::distance(it1, it2): возвращает количество элементов между двумя итераторами

Типы данных: для хранения суммы и длины используется long long для предотвращения переполнения при больших значениях.

Возвращаемое значение: функция search_subsequence возвращает std::vector<int>, содержащий найденную последовательность. При пустом входном массиве возвращается пустой вектор.

Пример работы алгоритма

Массив: 1 2 3 0 4 5 6 0

Проход:
- 1 < 2: последовательность [1,2], длина 2, сумма 3
- 2 < 3: последовательность [1,2,3], длина 3, сумма 6
- 3 > 0: сброс, последовательность [0], длина 1, сумма 0
- 0 < 4: последовательность [0,4], длина 2, сумма 4
- 4 < 5: последовательность [0,4,5], длина 3, сумма 9
- 5 < 6: последовательность [0,4,5,6], длина 4, сумма 15
- 6 > 0: сброс, последовательность [0], длина 1, сумма 0

Результат: [0,4,5,6] (длина 4, сумма 15)

Альтернативные последовательности: [1,2,3] имеет длину 3, что меньше 4, поэтому не выбирается.

Критерий выбора: при равенстве длин выбирается последовательность с большей суммой.

Например, для массива [1,2,3,10,20,30]:
- последовательность 1: [1,2,3], длина 3, сумма 6
- последовательность 2: [10,20,30], длина 3, сумма 60
- выбирается [10,20,30] (сумма больше)
